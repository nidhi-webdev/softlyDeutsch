## 📱 RESPONSIVE DESIGN

### Desktop (>1024px)
```
Navigation:  Full menu visible
Hero:        Left content + Right 3 cards
Skills:      4 columns in a row
Levels:      3 columns in a row
Testimonials: 3 columns in a row
```

### Tablet (768-1024px)
```
Navigation:  Full menu visible
Hero:        Stacked, still shows cards
Skills:      2 columns
Levels:      3 columns or 2+1
Testimonials: 2-3 columns
```

### Mobile (<768px)
```
Navigation:  Hamburger menu (expands on click)
Hero:        Stacked vertically
Skills:      1 column
Levels:      1 column
Testimonials: 1 column (scroll)
Buttons:     Full width
```

## ⚡ KEY INTERACTIONS

### Mobile Hamburger Menu
```
Click [☰] on mobile
  ↓
Menu expands
  ↓
Shows: Features, Pricing, Testimonials
  ↓
Shows: Sign In, Get Started
  ↓
Click any item → Navigation or Click [✕] → Close
```

### Hover Effects
```
Buttons:      Color change
Cards:        Shadow increase + slight scale
Skill Cards:  Scale up + shadow increase
Level Cards:  Scale up + shadow increase
Links:        Underline/color change
```

```
Desktop/Tablet: Smooth scroll to sections

               Content scrolls below nav
```
