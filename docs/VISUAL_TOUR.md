## 🔄 User Interaction Flow

### Scenario 1: First Time User
```
1. Visit landing page (/)
   ↓
2. Click "GET STARTED"
   ↓
3. Redirected to /dashboard
   ↓
4. See 4 categories
   ↓
5. Click "🔤 NOUNS" category
   ↓
6. See 4 lessons in grid
   ↓
7. Click "START LESSON" on first lesson
   ↓
8. Read lesson content & do exercises
   ↓
9. Complete lesson → marked as done
   ↓
10. Return to dashboard, progress updated
```

### Scenario 2: Switching Categories
```
1. User on dashboard
   ↓
2. Currently viewing 🔤 NOUNS lessons
   ↓
3. Click 📚 GRAMMAR button
   ↓
4. Page smoothly transitions
   ↓
5. Grid shows 4 grammar lessons
   ↓
```

## 🎓 Lesson Content Example

### Lesson: "Nominativ - Der, Die, Das Grundlagen"
```
TITLE: 🔤 Nominativ: Der, Die, Das - Grundlagen
DURATION: 15 minutes
SKILL: Lesen (Reading)
LEVEL: A1

CONTENT SECTION:
═════════════════════════════════════════════════════════

Nominativ: Der, Die, Das

German nouns have three genders. Learning the article 
with the noun is essential!

DIE DREI ARTIKEL (The three articles):

┌─────────────┬──────────┬──────────┬─────────────┐
│ Gender      │ Article  │ Example  │ Meaning     │
├─────────────┼──────────┼──────────┼─────────────┤
│ Masculine   │ der      │ der Mann │ the man     │
│ Feminine    │ die      │ die Frau │ the woman   │
│ Neuter      │ das      │ das Kind │ the child   │
└─────────────┴──────────┴──────────┴─────────────┘

Plural (all genders):
die (plural) - die Männer, die Frauen, die Kinder

═════════════════════════════════════════════════════════

EXERCISES:
───────────────────────────────────────────────────────

Exercise 1: Multiple Choice
Question: What does "Guten Morgen" mean?
Options:
  ○ Good morning
  ○ Good afternoon
  ○ Good evening
  ○ Good night

Exercise 2: Fill-in-the-Blank
Question: Fill in: "___ Kind ist klein."
Hint: Use the correct neuter article
Answer: [das]
Explanation: Das is the neuter article.

═════════════════════════════════════════════════════════
```

## 🎨 Visual Hierarchy

### Typography
```
Heading 1 (h1):   32px, Bold       - Page titles
Heading 2 (h2):   24px, Bold       - Section titles
Heading 3 (h3):   20px, Semi-bold  - Subsections
Body (p):         16px, Regular    - Main content
Small (small):    14px, Regular    - Captions
Tiny (span):      12px, Regular    - Badges, tags
```

### Spacing
```
Extra Small:   4px   - Component internal padding
Small:         8px   - Related elements spacing
Medium:        16px  - Section spacing
Large:         24px  - Major section spacing
Extra Large:   32px  - Page margins
```

### Shadows
```
None:          - Flat buttons, backgrounds
Subtle:        1px blur - Subtle depth
Medium:        4px blur - Card default
Strong:        8px blur - Hover, active states
Very Strong:   16px blur - Modal overlays (future)
```
